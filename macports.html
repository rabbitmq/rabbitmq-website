This page describes how to install the RabbitMQ server under Mac OS X using MacPorts.

MacPorts is an easy-to-use system for installing open-source software onto Mac OS X computers. The MacPorts community has packaged thousands of projects as ports, including the RabbitMQ server. So once you have installed MacPorts, installing the RabbitMQ server takes just a single command: 

sudo port install rabbitmq

See the MacPorts website for more information about MacPorts.

RabbitMQ latest release repository

When a new version of the RabbitMQ server is released, it can take some time for the version included in the main collection of MacPorts packages to be updated. So we provide a special-purpose MacPorts repository to give OS X users easy access to the very latest release of the RabbitMQ server.

Once you have installed MacPorts, to add our repository do the following:

  1. Open /opt/local/etc/macports/sources.conf in a text editor, and add the following http: URL before the existing rsync: URL as follows: 
    http://www.rabbitmq.com/releases/macports
rsync://rsync.macports.org/release/ports/ [default]
  2. Use the port sync command to download the indices from the newly-added repository: 
    sudo port sync
